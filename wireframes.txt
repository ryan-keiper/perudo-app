=======================================================
INTERACTIVE ELEMENTS & BEHAVIORS
=======================================================

MAIN HUB:
- Buttons: Scale on tap, haptic feedback
- Leaderboard: Pull to refresh, tap to view profile
- Settings: Slide in from right

GAME LOBBY:
- Player cards: Show join/leave animations
- Copy code: Show toast confirmation
- Ready toggle: Shake animation when pressed
- Start button: Pulse when all ready

GAME BOARD:
- Player avatars: Pulse on turn, dim when eliminated
- Dice: Shake animation when revealed
- Bid panel: Number steppers with haptic feedback
- Challenge: Confirmation modal with dramatic animation
- Results: Dice count-up animation with sound

=======================================================
RESPONSIVE CONSIDERATIONS
=======================================================

375px (Mobile):
- Single column layout
- Touch-friendly 44px minimum tap targets
- Large text for readability
- Simplified player circle

768px (Tablet):
- Expanded player circle with more details
- Side-by-side action panels
- Larger dice visualization
- Chat panel option

=======================================================
ACCESSIBILITY NOTES
=======================================================

- High contrast ratios (navy/pearl, crimson/pearl)
- Large text sizes (minimum 16px)
- Icon + text labels for clarity
- Voice-over friendly dice descriptions
- Color-blind friendly (patterns + colors)
- Haptic feedback for important actions
